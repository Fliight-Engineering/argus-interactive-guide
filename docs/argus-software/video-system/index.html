Video System

The Argus video system provides real-time streaming from multiple remote cameras with flexible viewing options.

Main Toolbar​

The toolbar at the top of the screen provides quick access to all main functions:

Main toolbar with recording, gallery, voice, grid view, and settings controls

ButtonFunction
RecordingStart/stop video recording for all cameras
GalleryOpen the recordings folder
VoiceOpen the Voice System Control panel
Grid ViewToggle between 1+3 and 2+2 camera layouts
SettingsOpen application settings
MinimizeMinimize the application window
CloseExit the application

Video View Switching​

Toggle between two camera layout options using the Grid View button in the toolbar.

1+3 Layout​

One large main view + three smaller views on the right. This is the default layout when the application starts.

1+3 layout: Main camera view with three secondary views on the right

2+2 Layout​

Four equal-sized camera views in a grid format.

2+2 layout: Four camera views in equal grid

tip

Double-click any camera view to make it the main (large) view in 1+3 layout.

Video Recording Function​

Click the Recording button in the toolbar to start recording all camera feeds simultaneously.

Click the Recording button to begin capturing all camera feeds

When recording is active:

Recording indicator showing active recording with duration timer

  • Recording Indicator: A red dot appears when recording is active
  • Duration Timer: Shows elapsed recording time
  • Stop Recording: Click the Recording button again to stop

All connected camera feeds are recorded simultaneously in sync, ensuring comprehensive coverage for post-training review.

Video Recording Gallery Access​

Click the Gallery button in the toolbar to open the recordings folder.

Click the Gallery button to access saved recordings

Recordings are stored in the directory specified in Settings. Use recordings for:

  • Post-training review and analysis
  • Trainee coaching and feedback
  • Assessment and performance tracking
