# Tổng Quan

Phần này hướng dẫn bạn qua quá trình cài đặt và vận hành hoàn chỉnh hệ thống Argus VTS. Nếu lên kế hoạch đúng cách, việc cài đặt chỉ mất khoảng **5 phút**.

### Khởi Tạo Nút Radio

<div class="card-content">

</div>

</a>

<a href="./cameras.html" class="setup-overview-card">

### Camera

<div class="card-content">

Bật và đồng bộ hệ thống camera từ xa sau khi điểm truy cập đã tạo mesh.

</div>

</a>

<a href="./tablet-setup.html" class="setup-overview-card">

### Cài Đặt Máy Tính Bảng

<div class="card-content">

Cấu hình máy tính bảng huấn luyện viên ở chế độ ngang và kết nối với mạng VTS.

</div>

</a>

<div class="card-content">

Gắn nút radio máy lên xe và định vị camera để có góc nhìn đào tạo tối ưu.

</div>

</a>

</div>

</a>
