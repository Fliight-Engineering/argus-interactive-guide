# Cámaras

Una vez que los nodos de red estén conectados en red mesh (mostrando verde fijo), encienda las cámaras de la siguiente manera:

<div class="highlights-grid-single">

<div class="highlight-card">

### Encendido de las cámaras

<div class="card-content">

<table class="image-info-card">
<tr>
<td class="image-info-card-text">

1. Encienda las cámaras de la misma manera mediante el botón de encendido negro
2. El anillo LED de estado del botón de la cámara se iluminará en **rojo** al encenderse
3. Se escuchará un **leve clic** mientras el objetivo se calibra
4. La cámara estará lista una vez que se complete la calibración (los clics se detienen)
