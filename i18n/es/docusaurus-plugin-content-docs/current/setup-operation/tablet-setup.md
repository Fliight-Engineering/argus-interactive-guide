### Inicio de sesión

<div class="card-content">

#### Inicio de sesión con PIN

Para iniciar sesión en la tablet:

1. **Deslice hacia arriba** desde la pantalla de bloqueo para mostrar la página de inicio de sesión del usuario de Argus VTS
2. **Toque el campo de entrada del PIN**
3. **Ingrese el PIN de la tablet Windows**: `2580`
4. **Presione Enter** o toque Aceptar

El escritorio estará listo para su uso.

:::info PIN de la tablet Windows
El PIN predeterminado para la tablet Argus VTS es: **2580**
:::

#### Después de iniciar sesión

Una vez que haya iniciado sesión:

1. El escritorio debería ser visible
2. El software Argus debería estar disponible (verificar en la barra de tareas)
3. El WiFi debería haberse conectado automáticamente; si no, consulte la sección a continuación.

### Configuración del WiFi

<div class="card-content">

Conecte la tablet del instructor a la red WiFi utilizando:

- **Nombre de red WiFi**: `ARGUS-VTS-XXX*` *(XXX es el número del kit)*
- **Contraseña**: `BarmincoVTS`

El software Argus se conectará automáticamente a las 4 cámaras — no se requiere configuración adicional.

Una vez conectado:
1. Inicie el software Argus (generalmente anclado a la barra de tareas o al menú de inicio)
2. Espere a que el software se inicialice (10-15 segundos)
3. El software debería detectar automáticamente las cámaras conectadas
4. Las 4 transmisiones de cámara deberían comenzar a reproducirse automáticamente
