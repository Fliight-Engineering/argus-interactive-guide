### Configuración del nodo de máquina

<div class="card-content">

El nodo de máquina está diseñado para proporcionar una colocación óptima de las antenas durante la operación, pero se empaca de forma compacta para su almacenamiento; por lo tanto, requiere una configuración física sencilla para su despliegue. Los pasos son los siguientes:

1. Saque el nodo de máquina del maletín rígido y despliegue las dos antenas inferiores

2. Desenganche el mástil de antenas y gírelo hacia la posición vertical

3. Tire del lazo del pestillo y enganche el pasador de bloqueo una vez que el mástil esté en posición vertical

4. Gire dos de las antenas de modo que sus codos apunten hacia arriba y pliegue todas las antenas en su posición de enclavamiento a 45 grados

El nodo de máquina ya está listo para montarse y encenderse.

### Encendido y orden de inicio de los nodos de red

<div class="card-content">

<table class="image-info-card">
<tr>
<td class="image-info-card-text">

Siempre encienda los nodos de red PRIMERO antes de encender otros componentes del sistema.

Para encender los nodos de red:

1. Presione firmemente el botón de encendido negro grande
2. Suelte el botón
3. Espere a que el nodo de red se inicialice (observe el estado del LED)

</td>

### Indicadores LED de estado

<div class="card-content">

<table class="image-info-card">
<tr>
<td class="image-info-card-text">

El nodo de red pasará de azul a verde a medida que se conecte a la red:

| Estado del LED | Significado | Acción requerida |
|------------|---------|-----------------|
| **Colores RGB pulsantes** | Inicializando | Espere — el sistema está arrancando |
| **Azul fijo** | Listo | Unidad encendida, esperando conexión |
| **Verde fijo** | Conectado | Unido en red mesh con otros nodos — listo para operar |

</td>
