# Sistema de Actualización

Argus incluye un sistema de actualización automática que mantiene tu software al día con las últimas funciones y mejoras.

## Control de Versiones

La versión actual del software se muestra en la esquina superior izquierda de la aplicación, junto al logotipo. Esta insignia de versión muestra:

<div class="image-wrapper-30">
  ![Version Badge](/img/vts-guide/version-badge.png)
  *Insignia de versión que muestra la versión actual del software*
</div>

- La versión instalada actualmente (p. ej., **v0.0.28**)
- Haz clic para buscar actualizaciones

## Proceso de Actualización

Cuando hay una nueva versión disponible:

<div class="image-wrapper-70">
  ![Update Available Dialog](/img/vts-guide/update-available.png)
  *Notificación de actualización que muestra la versión actual y la más reciente con opciones de descarga*
</div>

1. **Notificación**: Aparece una notificación indicando que hay una actualización disponible
2. **Descarga**: Haz clic para descargar el paquete de actualización
3. **Progreso**: Una barra de progreso muestra el estado de la descarga con la velocidad y el tiempo restante
4. **Instalación**: Una vez descargada, haz clic en **Install** para aplicar la actualización
5. **Reinicio**: La aplicación se reiniciará con la nueva versión

:::note
Las actualizaciones se descargan desde servidores en la nube seguros. Se requiere conexión a internet para verificar y descargar actualizaciones.
:::
