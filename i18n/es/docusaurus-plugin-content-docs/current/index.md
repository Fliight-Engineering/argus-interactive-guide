# Argus VTS Guide
# Guía del Usuario

Bienvenido a la guía del usuario del Sistema de Capacitación por Video Argus (VTS). Esta guía le ayudará a desplegar, probar y utilizar el Argus VTS.

Desarrollado originalmente para capacitar de forma remota a operadores de vehículos pesados subterráneos, Argus funciona igualmente bien para cualquier situación en la que necesite transmisiones de video en vivo desplegadas rápidamente. Todo lo necesario viene en maletines resistentes diseñados para una configuración rápida en cualquier lugar.
