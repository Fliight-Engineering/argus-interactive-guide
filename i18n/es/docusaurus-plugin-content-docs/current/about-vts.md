# Acerca del VTS

## Descripción General

Esta guía le ayudará a desplegar, probar y utilizar el Sistema de Capacitación por Video Argus (VTS). Desarrollado originalmente para capacitar de forma remota a operadores de vehículos pesados subterráneos, Argus funciona igualmente bien para cualquier situación en la que necesite transmisiones de video en vivo desplegadas rápidamente. Todo lo necesario viene en maletines resistentes diseñados para una configuración rápida en cualquier lugar.

---

## Componentes Principales del Sistema

El Sistema de Capacitación por Video Argus comprende cuatro componentes principales:

| **Componente** | **Descripción** |
|-----------|-------------|
| **El Argus Software** | Plataforma de software basada en Windows que otorga a los instructores remotos acceso a múltiples transmisiones de video desde cámaras remotas |
| **Los Nodos de Red Argus (AP)** | Nodos de red inalámbrica que trabajan en conjunto para crear una red mesh |
| **Sistemas de Cámara Remota (RCS)** | Cámaras de transmisión de video portátiles que se conectan rápidamente a los vehículos de entrenamiento |
| **Tablet PC del Instructor** | Se conecta a los nodos de red para ver las transmisiones de cámara y guardar grabaciones |
