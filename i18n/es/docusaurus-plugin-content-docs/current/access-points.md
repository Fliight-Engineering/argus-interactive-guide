# Nodos de Red

## Descripción General

Los nodos de red son dispositivos de red inalámbrica que trabajan en conjunto para crear una red mesh. Diseñados para transmisión de video confiable en ubicaciones subterráneas o sin conexión donde el Wi-Fi estándar no está disponible.

:::info ¿Qué es una Red Mesh?
Una red mesh consiste en múltiples nodos de red que trabajan juntos para extender la cobertura inalámbrica. Cada nodo de red puede retransmitir datos de otro, creando una red autorreparable que se adapta a obstáculos y al terreno.
:::
