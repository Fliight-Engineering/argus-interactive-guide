Nodos de red: Despliegue

Los nodos de red crean una red mesh dinámica que se autoconfigura durante la instalación.

Los nodos repetidores están pensados para situaciones en las que se necesita crear una ubicación segura para el instructor a una distancia significativa del área operativa del vehículo (200 m o más) o al doblar una esquina.

Información de montaje​

Los nodos repetidores tienen patas magnéticas que también funcionan como ganchos de montaje para fijarse a superficies de malla.

Opciones de montaje​

  • Placas de pernos: Fijación a placas metálicas con pernos
  • Fijación en malla: Enganche directamente en malla metálica soldada

Requisitos de posicionamiento​

  • Altura: Coloque a una altura igual o superior a la del nodo de máquina
  • Línea de visión: Posicione con vista despejada hacia los demás nodos de red — evite depresiones en la pared o en el camino que puedan obstruir la línea de visión
  • Ubicación: Cuando sea necesario, posicione el nodo repetidor en el punto medio del área operativa para ampliar la cobertura de la red según sea necesario

Nodo de red: Orientación de las antenas​

Las antenas del nodo de red pueden configurarse para proporcionar el máximo alcance a lo largo de una galería recta o para proporcionar señal alrededor de las curvas de la galería:

Ambas antenas hacia arriba para el máximo alcance directo a lo largo de una galería larga

Una antena hacia arriba y la otra hacia abajo, para proporcionar señal al doblar una curva

Configuración de nivel para el entrenamiento​

Para planificar el entrenamiento con el VTS, el diagrama de mapa a continuación muestra un ejemplo de configuración de nivel donde:

  • Posición del instructor situada fuera del recorrido de la cargadora y otros vehículos
  • Ubicaciones del nodo del instructor y el nodo repetidor mostradas posicionadas a lo largo de una galería larga y al doblar una curva

Se puede lograr un alcance operativo significativo siguiendo las instrucciones de configuración de esta guía
